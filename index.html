This is a web page created for the course Internet Programing Environments.


While creating this website was fairly enjoyable, there were some challenging aspects. First and foremost, every aspect of programming in a web environment is different than programing locally. Debugging is different, as more tests are done by displaying the results in a browser, javascript's asyncronyous calls provide insight as to why things do not work as well as being different from the programing languages we use often.


The website was built using html, javascript and jquery. The main use of jquery was to validate forms, and perform their actions. Templates were not used at all.
